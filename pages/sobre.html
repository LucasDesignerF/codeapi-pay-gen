Sobre o Projeto

O CodeAPI - Payment Links Gen é uma ferramenta desenvolvida para simplificar a criação de links de pagamento via Pix. Nosso objetivo é permitir que qualquer pessoa gere links personalizados de forma rápida e prática, sem complicações.

Este projeto foi idealizado e desenvolvido por LucasDev, sob a marca CodeProjects. Utilizamos tecnologias modernas como HTML, TailwindCSS e JavaScript para garantir uma experiência fluida e eficiente.

Para dúvidas, sugestões ou suporte, junte-se ao nosso servidor no Discord: CodeProjects Discord.
